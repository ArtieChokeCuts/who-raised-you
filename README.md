# Who Raised You

Playable music video prototype for the 2:19 track.

## Play

GitHub Pages deploys from `main` with the included workflow:

https://artiechokecuts.github.io/who-raised-you/

## Controls

Desktop: `A/D` or arrows to move, `W`/`Up`/`Space` to jump, `S`/`Down` to duck.

Phone: use the on-screen left/right, jump, and duck buttons.

## Note

The checked-in Pages build is playable even before binary assets are uploaded. If the MP3 is missing, it runs on a silent 2:19 master clock so the dodge-and-trap loop still works.
